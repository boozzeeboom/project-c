Scene System Architecture - CORRECTED

Date: 29.04.2026 | Status: ARCHITECTURE FIXED

Problem Summary

Scene system integration failed - player didn't spawn, errors about missing SceneTransitionCoordinator

Root Cause: Unnecessary Middle Layer

BEFORE (Broken): ServerSceneManager → ClientRpc → SceneTransitionCoordinator → ClientSceneLoader AFTER (Fixed): ServerSceneManager → ClientRpc → ClientSceneLoader (direct)

Architecture Diagram

┌─────────────────────────────────────────────────────────────┐ │ BootstrapScene.unity │ │ │ │ NetworkManager (GameObject "NetworkManager") │ │ ├── Unity.Netcode.NetworkManager │ │ └── NetworkManagerController │ │ │ │ Runtime (GameObject "Runtime") │ │ ├── ClientSceneLoader (DontDestroyOnLoad) │ │ │ └── OnClientConnectedCallback → AutoLoadScene() │ │ │ │ │ └── ServerSceneManager (NetworkBehaviour) │ │ └── OnClientConnected → FindPlayer + SendInitialScene │ │ └── Direct ClientRpc to ClientSceneLoader │ │ │ │ PlayerSpawner (GameObject "PlayerSpawner") │ │ ├── NetworkObject │ │ ├── NetworkPlayerSpawner (MonoBehaviour) │ │ └── NetworkPlayer │ │ │ └─────────────────────────────────────────────────────────────┘ ┌─────────────────────────────────────────────────────────────┐ │ WorldScene_X_Y.unity (additive loaded) │ │ │ │ WorldRoot_X_Y │ │ ├── DirectionalLight │ │ ├── GroundPlane_X_Y │ │ └── Boundaries_X_Y │ └─────────────────────────────────────────────────────────────┘

Host Start Flow

1. User clicks "Host" → NetworkManagerController.StartHost() 2. NetworkManager.StartHost() fires OnClientConnectedCallback(0) 3. NetworkPlayerSpawner.Update() → SpawnLocalPlayer() 4. NetworkObject.SpawnAsPlayerObject(0) → NetworkPlayer.OnNetworkSpawn() 5. ClientSceneLoader.OnClientConnectedCallback(0) → AutoLoadInitialScene() 6. LoadSceneWithNeighbors(SceneID(0,0)) 7. Player appears in world

Fixes Applied

FIX-001: ServerSceneManager - Direct RPC

var coordinator = NetworkManager.Singleton.GetComponent(); if (coordinator != null) coordinator.HandleInitialScene(scene); var loader = FindAnyObjectByType(); if (loader != null) loader.LoadScene(scene, spawnPosition);

FIX-002: ClientSceneLoader - Auto-load

private void OnEnable() { NetworkManager.Singleton.OnClientConnectedCallback += OnClientConnectedCallback; } private void OnClientConnectedCallback(ulong clientId) { if (clientId == NetworkManager.Singleton.LocalClientId && _currentScene.Equals(default)) { StartCoroutine(AutoLoadInitialSceneCoroutine()); } }

FIX-003: NetworkPlayerSpawner - Update timing

private bool _hasSpawnedHostPlayer = false; private void Update() { if (useScenePlayerAsHost && !_hasSpawnedHostPlayer && NetworkManager.Singleton != null && (NetworkManager.Singleton.IsHost || NetworkManager.Singleton.IsServer)) { _hasSpawnedHostPlayer = true; SpawnLocalPlayer(); } }
