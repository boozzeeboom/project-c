# Система инвентаря и подбора предметов

**Последнее обновление:** 4 апреля 2026 г. | **Версия:** `v0.0.6-inventory`

---

## 📋 Обзор

Система подбора предметов и кругового инвентаря в стиле GTA 5. Предметы группируются по типам в 8 секторах колеса. При наведении на сектор с несколькими предметами раскрывается список.

---

## 🏗️ Архитектура

### Компоненты

| Файл | Роль |
|------|------|
| `ItemType.cs` | Enum типов (Type1–Type8) + ItemData ScriptableObject |
| `Inventory.cs` | Singleton-менеджер: группирует предметы по типам |
| `PickupItem.cs` | Подбираемый объект в мире (триггер, покачивание) |
| `ItemPickupSystem.cs` | Обработка клавиши E: поиск ближайшего → подбор |
| `InventoryUI.cs` | Круговое колесо: 8 секторов, GL-рендер, hover, подсписки |
| `ControlHintsUI.cs` | Подсказки E и Tab в HUD |

### Поток данных

```
Игрок (E) → ItemPickupSystem → PickupItem.Collect()
    ↓
Inventory.AddItem(ItemData)
    ↓
InventoryUI: обновляет цвет сектора + счётчик
    ↓
Hover на сектор → DrawSublist (если count > 1)
```

---

## 🎮 Управление

| Клавиша | Действие | Режим |
|---------|----------|-------|
| **E** | Подобрать ближайший предмет (радиус 3м) | Пеший |
| **Tab** | Открыть/закрыть круговой инвентарь | Пеший |

## 📐 Круговое колесо

### Сектора (8 штук)

| Index | Позиция | Тип | Цвет (есть предмет) |
|-------|---------|-----|---------------------|
| 0 | Верх | Тип 1 | Зелёный |
| 1 | Верх-право | Тип 2 | Зелёный |
| 2 | Право | Тип 3 | Зелёный |
| 3 | Низ-право | Тип 4 | Зелёный |
| 4 | Низ | Тип 5 | Зелёный |
| 5 | Низ-лево | Тип 6 | Зелёный |
| 6 | Лево | Тип 7 | Зелёный |
| 7 | Верх-лево | Тип 8 | Зелёный |

### Логика отображения

- **Пустой сектор:** тёмно-серый (`emptyColor`)
- **Есть предметы:** зелёный (`hasItemsColor`)
- **Наведение мыши:** жёлтая подсветка (`hoverColor`)
- **>1 предмет в секторе:** при наведении появляется подсписок с названиями

## 📦 Создание предмета

### 1. Создать ItemData (ScriptableObject)

```
Project окно → Create → Project C → Item Data
```

Поля:
- **Item Name** — отображаемое имя
- **Item Type** — тип (Type1–Type8)
- **Description** — описание
- **Icon** — иконка (опционально)

### 2. Разместить в мире

1. Создать 3D-объект (Sphere/Cube/etc.)
2. Добавить компонент `PickupItem`
3. Перетащить ItemData в поле **Item Data**
4. Настроить `floatSpeed` и `floatAmplitude` (покачивание)
