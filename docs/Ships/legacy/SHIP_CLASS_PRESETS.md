## 🎮 Как Использовать

### В Unity Editor

```
1. Выбрать корабль в Hierarchy
2. Inspector → ShipController
3. Секция "Класс Корабля"
4. Ship Class: выбрать из dropdown (Light/Medium/Heavy/HeavyII)
5. Параметры автоматически обновятся
```

### В Коде

```csharp
// При создании корабля
shipController.SetShipClass(ShipClass.Heavy);

// Или через Inspector (автоматически применяется в OnValidate)
```

---

## 🔧 Настройка Под Себя

Если пресеты не подходят — можно вручную настроить параметры в Inspector:

1. Выбрать корабль
2. Inspector → ShipController
3. Изменить любой параметр (Yaw Force, Smooth Time и т.д.)
4. Изменения сохранятся и переопределят пресет

**Совет:** Начни с пресета Medium, затем微调 по ощущениям.
