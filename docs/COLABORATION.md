## 🌍 О проекте

**Project C: The Clouds** — приключенческая MMO во вселенной «Интеграл Пьявица» Бруно Арендта.

Проект разрабатывается **своими силами** — небольшой командой энтузиастов. Код открыт для сообщества, чтобы мотивировать всестороннее развитие: модификации, тулы, изучение.

**Стек:** Unity 6 + URP + Netcode for GameObjects (NGO) | C#

**Статус:** Активная разработка (v0.0.40+). Реализованы NPC-квесты, крафт, майнинг, инвентарь, корабли, смена дня/ночи и многое другое. Полный roadmap: [`docs/MMO_Development_Plan.md`](MMO_Development_Plan.md)

Отправляя Pull Request или ассет, вы автоматически соглашаетесь с условиями ниже.

Вы соглашаетесь, что ваш код распространяется под **MIT** на тех же условиях, что и остальной код проекта:

- Код лицензируется под MIT без дополнительных ограничений
- Вы сохраняете авторство (copyright), но даёте всем те же права, что и MIT

### Для ассетов (3D, текстуры, звуки, анимации, VFX)

Вы подтверждаете, что:

## 🚀 Как присоединиться

### 1️⃣ Напишите нам

**Telegram:** [@indeed174](https://t.me/indeed174) — расскажите, кто вы, чем хотите помочь, какой у вас опыт.

**Либо** создайте Issue на GitHub с меткой `contributor`:
[github.com/boozzeeboom/project-c/issues](https://github.com/boozzeeboom/project-c/issues)

### 2️⃣ Форкните репозиторий

```bash
git clone https://github.com/boozzeeboom/project-c.git
cd project-c
git checkout -b feature/ваша-фича
```

### 3️⃣ Создайте Pull Request

Любой PR проходит ревью. Главные требования:
- Код следует существующему стилю (см. существующие файлы)
- Ассеты оптимизированы (poly count, texture size)
- **PR описывает, какой вклад вы делаете (код/ассет/документация)**
- **PR включает согласие с CLA** (см. [`PULL_REQUEST_TEMPLATE.md`](../.github/PULL_REQUEST_TEMPLATE.md))

### 4️⃣ После принятия

- Ваш код/ассет появляется в игре
- Вы получаете именную табличку / credits
- Вы в списке контрибьюторов

## 💬 FAQ

**❓ Проект продаётся?**
Нет. Игра бесплатная. Мы делаем её своими силами. Код открыт для модов и изучения.

**❓ Могу ли я сделать мод?**
Да! MIT-код позволяет создавать моды, расширения, тулы. Запрещено только распространение ассетов и контента игры.

**❓ Я отправил ассет — могу ли я потом его забрать?**
Нет, лицензия безотзывная. Мы будем использовать ассет в игре. Убедитесь, что вы готовы к этому.

**❓ Могу ли я использовать код в своём проекте?**
Да, код под MIT — можно в любых целях, коммерческих тоже. С указанием авторства.

**❓ Это оплачивается?**
Нет. Проект делается энтузиастами. Вознаграждение — опыт, портфолио, сообщество и именные ассеты в игре.

**❓ Нужно ли знать лор вселенной?**
Желательно — это помогает создавать атмосферные вещи. Начните с [`WORLD_LORE_BOOK.md`](WORLD_LORE_BOOK.md).

**❓ Сколько времени нужно уделять?**
Сколько хотите. Раз в неделю или каждый день — всё ценно. Мы не ставим дедлайнов.
