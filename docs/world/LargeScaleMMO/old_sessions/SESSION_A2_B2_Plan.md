## 🔧 Технические детали

### A2: Алгоритм генерации мешей

```
GeneratePeakMesh(PeakData peak):
  1. Определить тип меша по peak.shapeType:
     - Tectonic:   цилиндр 64×16, острые грани, ridge noise
     - Volcanic:   вытянутая сфера, кратер в вершине
     - Dome:       пологий купол
     - Isolated:   конус с широкой базой, крутая вершина

  2. Применить heightProfile (AnimationCurve):
     Для каждой вершины: y *= heightProfile.Evaluate(normalizedRadius)

  3. Применить keypoint deformation:
     Для каждого HeightmapKeypoint:
       локальное возмущение на normalizedRadius × noiseWeight

  4. Применить FBM noise:
     vertexOffset += FBM(vertexPosition * frequency) * amplitude * noiseWeight

  5. Пересчитать нормали: mesh.RecalculateNormals()

  6. Назначить материалы:
     - Выше snowLineY → Snow_Generic.mat
     - Ниже snowLineY → Rock_{Biome}.mat

  7. Добавить CapsuleCollider:
     - Высота = peak.worldPosition.y - основание
     - Радиус = peak.baseRadius * 0.8 (чуть меньше меша)
```
