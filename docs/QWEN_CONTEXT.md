## 🔴 Известные проблемы (нужно починить ПЕРВЫМ ДЕЛОМ)

### Disconnect кнопка в левом углу

**Описание:** Кнопка Disconnect создаётся программно в `NetworkUI.cs:CreateDisconnectButton()`, но позиционируется в левом нижнем углу экрана — почти невидима.

**Файл:** `Assets/_Project/Scripts/UI/NetworkUI.cs`, метод `CreateDisconnectButton()`

**Как починить:**
```csharp
// Вариант 1: Центр экрана через якоря
rt.anchorMin = new Vector2(0.5f, 0.5f);
rt.anchorMax = new Vector2(0.5f, 0.5f);
rt.pivot = new Vector2(0.5f, 0.5f);
rt.anchoredPosition = Vector2.zero;

// Вариант 2: Привязать к connectionPanel (он позиционирован правильно)
Transform parent = connectionPanel != null ? connectionPanel.transform : canvas.transform;
btnObj.transform.SetParent(parent, false);
```

**Подробнее:** `docs/NETWORK_ARCHITECTURE.md` — раздел "Известные проблемы и задачи"

##  Управление

| Клавиша | Действие | Режим |
|---------|----------|-------|
| WASD | Движение | Пеший + корабль |
| Space | Прыжок | Пеший |
| Shift | Бег / ускорение (корабль x2 тяга) | Оба |
| F | Сесть/выйти из корабля | Оба |
| E | Подобрать предмет / открыть сундук | Пеший |
| Tab | Круговой инвентарь | Пеший |
| Q/E | Лифт вверх/вниз | Корабль |
| Escape | Toggle Disconnect кнопка | Подключён |
