    {
        loader.LoadInitialScene(new ProjectC.World.Scene.SceneID(0, 0));

## Architecture Overview

### Scene System Flow

```
BootstrapScene (first in Build Settings)
├── NetworkManager + NetworkManagerController
├── Runtime
│   ├── ClientSceneLoader (DontDestroyOnLoad) - loads/unloads scenes
│   └── ServerSceneManager (NetworkBehaviour) - tracks player positions
├── PlayerSpawner (NetworkPlayer.prefab)
└── NetworkTestCanvas
    └── NetworkTestMenu (Host, Client, Server, Load World [0,0])

WorldScene_X_Y (24 scenes, additive loaded)
├── WorldRoot_X_Y (at world position X*79999, Z*79999)
├── DirectionalLight
├── GroundPlane
├── SceneLabel
└── Boundaries (North/South/East/West colliders)
```

### How to Test

1. **ProjectC → World → Generate Bootstrap Scene**
2. **ProjectC → World → Generate World Scenes** (24 scenes)
3. **File → Build Settings** - drag BootstrapScene to top (index 0)
4. **Play** - click "Host" then "Load World [0,0]"
5. **Player spawns** at WorldScene_0_0 center: (39999.5, 3000, 39999.5)
6. **Move around** - at boundary (~79000), ClientSceneLoader detects scene change
7. **ServerSceneManager** sends RPC → ClientSceneLoader loads new scene
