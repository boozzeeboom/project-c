## 🔧 Как протестировать в Unity

### Шаг 1: Открыть Unity
```
Открыть проект ProjectC_client в Unity 6
```

### Шаг 2: Проверить компиляцию
```
Подождать пока Unity перекомпилирует скрипты
Проверить Console — НЕ должно быть ошибок компиляции
```

### Шаг 3: Запустить PeakDataScaler
```
Menu: Tools → Project C → Scale Peak Data (V2)
Откроется окно "Peak Data Scaler V2"

В окне:
- Включить "Показать превью размеров"
- Проверить таблицу — все 29 пиков с размерами
- Нажать "Масштабировать ВСЕ пики (29) — V2"

В Console должно появиться:
[PeakDataScaler] Эверест: meshHeight=750, baseRadius=420, h/r=1.79
[PeakDataScaler] Лхоцзе: meshHeight=620, baseRadius=350, h/r=1.77
... (для всех 29 пиков)
```

### Шаг 4: Построить горы
```
Menu: Tools → Project C → Build All Mountain Meshes (V2)
Откроется окно "Mountain Mesh Builder V2"

В окне:
- Включить "Удалить существующие горы" (если есть старые)
- Назначить материалы (если не назначены автоматически):
  * Himalayan Rock → Rock_Himalayan.mat
  * Alpine Rock → Rock_Alpine.mat
  * African Rock → Rock_African.mat
  * Andean Rock → Rock_Andean.mat
  * Alaskan Rock → Rock_Alaskan.mat
  * Snow → Snow_Generic.mat
- Нажать "Построить ВСЕ горы (29 пиков) — V2"

В Console должно появиться:
[MountainMassifBuilderV2] Эверест (Tectonic): baseY=0, topY=750 | meshH=750, baseR=420, h/r=1.79 | ...
[MountainMassifBuilderV2] Лхоцзе (Tectonic): baseY=0, topY=620 | meshH=620, baseR=350, h/r=1.77 | ...
... (для всех 29 пиков)
```

### Шаг 5: Проверить в Scene view
```
1. Найти GameObject "Mountains" в Hierarchy
2. Развернуть → 5 массивов → 29 пиков
3. Выбрать пик (например Эверест)
4. Проверить в Inspector:
   - MeshFilter.sharedMesh — есть меш
   - MeshCollider.sharedMesh — есть меш (convex=false)
   - MeshRenderer.sharedMaterial — назначен материал

5. В Scene view:
   - Проверить размер пиков (400-800 units tall)
   - Проверить формы (горные, НЕ "пипки")
   - Проверить h/r ratio (1.5-2.0)
   - Проверить что ВСЕ пики на земле (base Y=0)

6. Проверить коллайдеры:
   - Включить Gizmos → MeshCollider
   - Коллайдер должен соответствовать форме горы
```

### Шаг 6: Проверить в Play mode
```
Нажать Play
Проверить:
- FPS >= 60
- Горы рендерятся правильно
- Коллизии работают (корабль НЕ проходит сквозь горы)
- Нет ошибок в Console
```

## ✅ Критерии успеха

### Минимальные требования:
- [ ] Все 29 пиков построены БЕЗ ошибок компиляции
- [ ] Визуальный размер: 400-800 units tall (проверить в Scene view)
- [ ] Формы ГОРНЫЕ, НЕ "пипки/вытянутые капли"
- [ ] h/r ratio = 1.5-2.0 для большинства пиков
- [ ] MeshCollider соответствует форме горы (convex=false)
- [ ] Основание пиков на Y=0 (НЕ под землёй)

### Идеальный результат:
- [ ] Эверест выглядит как гора (высокий, с плечами, острая вершина)
- [ ] Кибо выглядит как вулкан (пологий, округлый)
- [ ] Денали выглядит как одинокая громада (массивные плечи)
- [ ] Все пики РАЗНЫЕ (НЕ одинаковые)
- [ ] FPS >= 60 в Play mode
- [ ] Пользователь доволен ✅✅✅

## 🚨 Если что-то НЕ работает

### Ошибки компиляции:
```
Проверить Console → скопировать ошибку
Обычно это:
- Missing namespace/import
- Typo в имени метода
- Несоответствие типов
```

### Пики НЕ строятся:
```
Проверить Console → ошибки при построении
Возможные причины:
- PeakData не заполнены (запустить PeakDataFiller сначала)
- Материалы не найдены (проверить пути)
- MountainMassif assets не найдены (проверить Data/World/Massifs/)
```

### Пики всё ещё маленькие:
```
Проверить что PeakDataScaler запущен
Проверить Console → логи масштабирования
Проверить Inspector пика → meshHeight/baseRadius значения
```

### Формы всё ещё ужасные:
```
Это КРИТИЧЕСКИЙ фидбек — сообщить:
- Что именно НЕ так (слишком острые? слишком круглые?)
- Какие пики выглядят плохо (все или конкретные?)
- Скриншоты если возможно
```

## 🎯 Что ожидать визуально

### Tectonic (Everest-style):
- exponent = 1.4 → concave profile (крутые склоны)
- Shoulder bulge на 35% высоты → характерное "плечо"
- Ridge noise → линейные хребты
- Asymmetry 12% → уникальный силуэт
- **Вид:** Острая вершина, крутые склоны, плечи, хребты

### Volcanic (Kilimanjaro-style):
- exponent = 0.65 → convex profile (пологие склоны)
- Shoulder bulge на 45% высоты → округлое утолщение
- Crater на вершине → впадина
- Asymmetry 6% → почти симметричный
- **Вид:** Пологий вулкан с кратером

### Dome (Foraker-style):
- exponent = 0.5 → очень convex (очень пологий)
- Shoulder bulge на 50% высоты → широкое утолщение
- No crater → плоская вершина
- Asymmetry 5% → симметричный
- **Вид:** Широкий купол, пологие склоны

### Isolated (Denali-style):
- exponent = 1.1 → slightly concave
- Shoulder bulge на 40% высоты → МАССИВНЫЕ плечи
- No ridge noise → гладкие склоны
- Asymmetry 20% → заметно асимметричный
- **Вид:** Одинокая громада с массивными плечами
