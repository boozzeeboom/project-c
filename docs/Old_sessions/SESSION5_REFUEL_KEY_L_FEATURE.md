# Фича: Кнопка дозаправки мезиевых паров (L)

**Сессия:** 5 | **Дата:** 12 апреля 2026 | **Приоритет:** P2
**Статус:** ✅ Реализовано

## Описание
Кнопка `L` для принудительной дозаправки мезиевых паров из атмосферы.
Работает с компромиссом — корабль теряет тягу и скорость на время дозаправки.

## Реализация
- **Кнопка:** `L` (зажми = дозаправка)
- **Скорость восстановления:** 2.0 fuel/s (в 6.6x быстрее idle regen 0.3)
- **Компромисс:** Пока идёт дозаправка:
  - thrust снижается на 50% (`thrustPenaltyDuringRefuel = 0.5`)
  - maxSpeed снижается на 30% (`speedPenaltyDuringRefuel = 0.7`)
- **Авто-стоп:** Когда fuel полный — дозаправка прекращается
