Мир

Итерации

Build → Core → Render → World → Network → Ships

Реализованные системы

✓ Volumetric cloud rendering (raymarch, Ghibli style)
✓ Window + Input (GLFW 3.4)
✓ Logger (spdlog), Config system
✓ ChunkManager, CircularWorld (350k radius)
✓ Network: Host/Client via ENet
✓ Player sync (position interpolation)
✓ Ship controls: W/S/A/D/Q/E/Z/X/C/V
✓ Jolt Physics integration

Стек технологий

GLFW 3.4 flecs ECS OpenGL 4.6 ENet 1.3.18 GLM spdlog Jolt Physics CMake

Миграция Unity → CLOUDENGINE

Unity MonoBehaviour → ECS System + Component
ScriptableObject → ECS Archetype / Data Asset
NetworkVariable → NetworkTransform + serialization
ServerRpc/ClientRpc → Custom Rpc system
