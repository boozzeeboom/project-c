## Основная премисса

В **1930 году** на Землю упали метеориты, принеся два вещества:

- **Антигравий** — металл с управляемым гравитационным полем
- **Мезий** — ядовитый газ, расщепляющий органические соединения

Катастрофа **«Мирового Падения»** сделала поверхность Земли непригодной для жизни. Человечество (~2 миллиона человек) переселилось **«над облаками»** — на платформы, закреплённые на горных вершинах.

Поверхность скрыта под **«Завесой»** — искусственным густым дымом, удерживающим пары мезия.

## Технологии

### Антигравий

| Свойство | Описание |
|----------|----------|
| **Что это** | Металл с собственным изменяемым гравитационным полем |
| **Происхождение** | Упал с метеоритами в 1930 году |
| **Свойство** | При подаче электромагнитного поля искажает гравитацию вокруг себя |
| **Дефицит** | Крайне дорогой, используется в сплавах с малым процентным содержанием |
| **Применение** | Судостроение, бижутерия, опорные платформы, все сферы жизни |

### Мезий

| Свойство | Описание |
|----------|----------|
| **Что это** | Вещество, расщепляющее органические соединения |
| **Температура кипения** | ~17°C |
| **Месторождения** | В нижних слоях атмосферы (под облаками) |
| **Применение** | Катализатор, топливо для генераторов МАГ/ГМАГ |

### Генераторы МАГ (Мезий-Антигравиевый Генератор)

| Тип | Назначение |
|-----|-----------|
| **МАГ-3** | Стандартные суда |
| **МАГ-11** | Усиленные суда |
| **ГМАГ** | Большегрузные суда |

**Принцип работы:** ротор вращается за счёт взаимодействия жидкого мезия с антигравиевым валом → создаёт постоянный электрический ток → ток подаётся на рамку-контур из антигравия → антигравитационное поле.

### СОЛ (Система Определения Личностей)

- Комплекс **гравитационного сканирования** для идентификации граждан
- Контроль миграции, учёт предпочтений
- **Операторы** — слепые/глухие люди (мозг лучше улавливает показания сканеров)
- Много «дыр» в системе

### ГРАДАР

- Комплекс определения координат физических тел в пространстве посредством **гравитационного сканирования**

### ЛРО (Лётный радиолокационный ответ)

- Обмен техническими характеристиками и положением подвесных конструкций летательного аппарата

### Другие технологии

| Технология | Описание |
|-----------|----------|
| **Пламб** | Пишущий предмет из свинцовых сплавов |
| **Старт-ключ** | Зелёная лампочка-стержень для запуска корабля |
| **Медальон-ключ** | Спиралевидный, пурпурный, открывает доступ к Люфтрейнджеру |
| **Препарат «ERREY»** | Экспериментальный препарат для реабилитации нервной системы |
| **ТПГ** | Топливный Паровой Генератор |
| **ПДТТ** | Паровой Двигатель Твёрдого Топла |

## Корабли

### Принцип работы

```
Мезий → МАГ (генератор) → Электричество → Рамка-контур (антигравий) → Невесомость
                                              ↓
                                    Ветровые лопасти → Движение
```

Корабль — это **рамка-контур из сплавов антигравия**, создающая антигравитационное поле. Ветровые лопасти (электродвигатели) обеспечивают горизонтальное движение.

### Классы кораблей

| Класс | Тип | Форма | Вместимость | Размеры | Скорость | Потребление (движение/дрейф) |
|-------|-----|-------|-------------|---------|----------|----------------------------|
| **Лёгкий** | Прогулочный | Торообразный, закрытый | 2-4 чел. | До 4м × 2м | ~350 км/ч | 10 кВт·ч / 50 Вт·ч |
| **Средний** | Пассажирский | Сигарообразный, крытый | До 13 чел. | До 13м × 3м | ~300 км/ч | 10 кВт·ч / 50 Вт·ч |
| **Тяжёлый I** | Пассажирский | Платформа, до 3 ярусов | 28 657 т | До 89м × 34м | 150 км/ч | 1 мВт·ч / 1 кВт·ч |
| **Тяжёлый II** | Грузовой | Открытая площадка, 2 яруса | 121 393 т | До 377м × 89м | 150 км/ч | 5 мВт·ч / 2,5 кВт·ч |

### Физика кораблей (из книги!)

**Что ХАРАКТЕРНО:**
- ✅ Плавное зависание на высоте
- ✅ Плавный набор/снижение высоты (изменение напряжения на рамке)
- ✅ Горизонтальное движение через ветровые лопасти
- ✅ Медленный разгон и торможение (масса + инерция)
- ✅ **Нет резких рывков** — баржи, не истребители

**Что НЕ характерно:**
- ❌ **Крен** (наклон боком) — рамка-контур стабилизирует
- ❌ Резкие манёвры — масса слишком велика
- ❌ Быстрые повороты — инерция

**Исключение:** военные/специальные суда (Гильдия Силы) могут иметь другие характеристики.

### Паромы

- **Маршрутный транспорт** между фермерскими поселениями и городами
- Малые паровые двигатели или МАГ
- Движение по системе **подвесных тросов**
- Максимальная скорость: **40 км/ч**

## Сюжетные линии

### 1. Линия Сарефа
Агент Гильдии Тайн с амнезией → ищет «Импенетрабеле» → манипулируется системой → попадает в культ фанатиков

### 2. Линия Рены и Дюка
Двое с амнезией просыпаются на заброшенном корабле → летят через весь мир → ищут свою идентичность

### 3. Линия Элли
Девочка-сирота → расшифровывает послания деда → раскрывает заговор правительства

### 4. Линия Бэнни и Уилла
Побег с фермы → экспериментальный препарат → подпольная сеть

### 5. Финал
Кора и Эмилия находят Люфтрейнджер → запускают систему очистки Земли → Завеса рассеивается

## Рекомендации для игры

### Визуальный стиль
- **Стимпанк + дизельпанк + пост-апокалипсис**
- Металлические платформы, ржавчина, облака
- Жёлто-зелёное освещение (атмосфера мезия)
- Гранитная и кирпичная архитектура городов
- **Ghibli-style:** мягкие закаты, объёмные облака, плавные контуры

### Биомы/Уровни
| Биом | Описание |
|------|----------|
| **Горные платформы** | Гималаи, Килиманджаро, Эльбрус — поселения на пиках |
| **4 Города НП** | Уникальная архитектура каждого города |
| **Облачная Завеса** | Фиолетовые молнии, ядовитые пары |
| **Фермерские угодья** | Разбросаны по горным пикам |
| **Заброшенные корабли** | Остовы, убежища, лут |
| **Люфтрейнджер** | Подземный комплекс очистки — финальная локация |

### Игровые механики (из книги!)
| Механика | Описание |
|----------|----------|
| **Полёт на кораблях** | Управление, навигация, стыковка (плавная физика — баржи, не истребители!) |
| **Исследование** | Города, фермы, заброшенные сооружения |
| **Стелс** | Избегание СОЛ, правительственных агентов |
| **Крафт** | Ремонт кораблей, создание устройств из запчастей |
| **Шифрование/дешифровка** | Головоломки с кодами (как у Элли) |
| **Ранговая система** | Продвижение по Гильдиям (кадет → суб-лейтенант → лейтенант) |
| **Ресурс-менеджмент** | Топливо (мезий), еда, деньги (МНП) |
| **Торговля** | Динамические цены между городами и фермами |
| **Фракции** | Репутация с Гильдиями и подпольными группировками |

### Важные замечания из книги для геймдизайна

1. **Корабли — НЕ истребители.** Это баржи с медленным разгоном. Без крена. Плавные.
2. **Мезий — не только топливо, но и опасность.** Спуск вниз = риск.
3. **СОЛ — система контроля.** Игрок может обходить её — это стелс-механика.
4. **Амнезия — центральная тема.** Потеряшки/алиенумы — особый класс NPCs.
5. **Люфтрейнджер — конечная цель.** Система очистки Земли — финальная миссия.
6. **Цифры буквами.** Так пишут в Новой Цивилизации — использовать в UI/диалогах.

## 📝 Примечания для разработки

### Сверка с текущей реализацией (SHIP_CONTROLLER)

| Элемент | В книге | В игре сейчас | Нужно исправить? |
|---------|---------|---------------|-------------------|
| Вертикальный подъём/спуск | ✅ Плавное изменение высоты | ✅ Q/E лифт вверх/вниз | ✅ Соответствует лору |
| Плавность | ✅ Медленно, как баржа | ⚠️ Дёрганое | ✅ Да — добавить smoothing |
| Banking | ❌ Нет | ⚠️ Планируется | ❌ Не добавлять |
| Стабилизация | ✅ Рамка-контур держит горизонт | ✅ Есть | ✅ Оставить |
| Топливо | ✅ Мезий | ❌ Нет | ✅ Добавить |
