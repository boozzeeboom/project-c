##  Лор мира

### Мир над облаками
Проект C происходит в мире, где земля скрыта под плотным облачным слоем. Горные пики выступают над облаками как острова в небе. Корабли — основной способ перемещения между этими островами.

### Типы кораблей
1. **Личные глайдеры** — лёгкие, одноместные, для коротких перелётов
2. **Торговые баржи** — медленные, грузоподъёмные, с каютами
3. **Пиратские клипперы** — быстрые, манёвренные, с вооружением
4. **Круизные дирижабли** — роскошные, медленные, многоместные

### Физика полёта в мире C
- **Антигравитация** — корабли используют «облачный двигатель» для компенсации гравитации
- **Ветровые течения** — между островами существуют воздушные потоки (будущая механика)
- **Турбулентность** — при приближении к облачному слою корабль трясёт
- **Высота** — максимальная высота ограничена «небесным барьером» (будущая механика)

## 🎮 Механики корабля (текущие)

### Управление
| Клавиша | Действие | Описание |
|---------|----------|----------|
| **W** | Тяга вперёд | Двигатель толкает корабль вперёд |
| **S** | Тяга назад | Торможение / задний ход |
| **A/D** | Рыскание (Yaw) | Поворот корпуса влево/вправо |
| **Q/E** | Крен (Roll) | Наклон корпуса влево/вправо |
| **Мышь Y** | Тангаж (Pitch) | Нос вверх/вниз |
| **Left Shift** | Ускорение | x2 тяга двигателя |

### Физика
| Параметр | Значение | Описание |
|----------|----------|----------|
| **Anti Gravity** | 1.0 | 0 = падает, 1 = зависает, 1.5 = поднимается |
| **Thrust Force** | 500 | Сила тяги двигателя |
| **Max Speed** | 30 м/с | Ограничение скорости |
| **Linear Drag** | 1.0 | Сопротивление воздуха |
| **Angular Drag** | 2.0 | Угловое сопротивление |
| **Stabilization** | 50 | Сила возврата к горизонту |

### Посадка/Взлёт
- **Посадка:** корабль может сесть на любую горизонтальную поверхность
- **Выход:** игрок выходит из корабля в пешем режиме (F)
- **Вход:** игрок подходит к кораблю и нажимает F

##  Механики (будущие)

### Топливо
- **Облачное топливо** — расходуется при полёте
- **Заправка** — на островах есть заправочные станции
- **Бак** — ограниченный объём, нужно планировать маршрут

### Повреждения
- **Столкновения** — корабль получает урон при столкновении
- **Ремонт** — на островах есть мастерские
- **Критические повреждения** — корабль не может лететь

### Апгрейды
- **Двигатель** — больше тяги, выше скорость
- **Бак** — больше топлива
- **Броня** — защита от столкновений
- **Грузовой отсек** — больше инвентаря

### Ветровые течения
- **Восходящие потоки** — поднимают корабль без расхода топлива
- **Нисходящие потоки** — тянут вниз, нужно компенсировать
- **Боковые ветры** — сносят корабль с курса

### Небесный барьер
- **Максимальная высота** — 10 000м над облаками
- **Эффекты** — турбулентность, потеря управления, урон
- **Визуал** — экран темнеет, облака сгущаются
