##  Ключевые элементы мира

### Антигравий

**Металл, противоречащий законам гравитации.**

- Попал на Землю в 1930 году с метеоритами
- Подвергается обработке — становится источником трансляции мощного поля
- Излучает антигравитационные волны
- **Ключевое свойство:** при подаче электромагнитного поля искажает гравитацию вокруг себя
- Дефицитен, крайне дорогостоящ
- Используется в малом процентном содержании в сплавах
- Активные свойства изменяются под воздействием электромагнитных сил

**Применение:** судостроение, бижутерия, опорные платформы, все сферы жизни

---

### Мезий

**Вещество, расщепляющее органические соединения.**

- Попало на Землю в 1930 году с ледяной оболочкой метеоритов
- Температура кипения — около 17°C
- **Месторождения:** в нижних слоях атмосферы (под облаками)
- Широко применяется как катализатор
- **Топливо для генераторов МАГ/ГМАГ**

---

### Завеса

**Искусственный густой дым, сдерживающий пары мезия.**

- Поднимается из нижних слоёв атмосферы (ядовитый газ)
- Производится заводами-генераторами (1950-е годы)
- Изначально создана для охлаждения мезийсодержащих соединений

---

### Новая Цивилизация

**Пост-апокалиптическое общество, выжившее над облаками.**

- Центр: четыре города Нового Правительства (НП)
- Фермерские угодья на верхушках гор (опорные платформы с антигравием)
- Валюта: Сколы / скольчата / жетоны / МНП (плоские металлические жетоны)

## 🚢 Корабли

### Принцип работы

Корабль — это **рамка-контур из сплавов антигравия**.

1. **Генераторы МАГ** вырабатывают электрический ток из мезия
2. Ток подаётся на **рамку-контур** из антигравия
3. Рамка создаёт **антигравитационное поле** — корабль зависает
4. **Ветровые лопасти** (электродвигатели) приводят в движение
5. Горизонтальная скорость задаётся рамкой-контуром и/или толкающими двигателями

```
Мезий → МАГ (генератор) → Электричество → Рамка-контур (антигравий) → Невесомость
                                              ↓
                                    Ветровые лопасти → Движение
```

### Физика кораблей (для геймплея)

**Что ХАРАКТЕРНО для кораблей:**
- ✅ Плавное зависание на высоте
- ✅ Плавный набор/снижение высоты (изменение напряжения на рамке)
- ✅ Горизонтальное движение через ветровые лопасти
- ✅ Медленный разгон и торможение (масса + инерция)
- ✅ Нет резких рывков — баржи, не истребители

**Что НЕ характерно (затравка для особого транспорта):**
- ❌ Крен (наклон боком) — рамка-контур стабилизирует
- ❌ Резкие манёвры — масса слишком велика
- ❌ Быстрые повороты — инерция

**Исключение:** военные/специальные суда (Гильдия Силы) могут иметь другие характеристики.

---

### Типы генераторов

| Тип | Назначение | Принцип |
|-----|-----------|---------|
| **МАГ** | Стандартные суда | Вращение ротора: жидкий мезий + антигравиевый вал |
| **ГМАГ** | Большегрузные суда | Усиленная версия МАГ |
| **Паровой двигатель** | Старые/дешёвые суда | Твердотопливный, для паромов |

**Отработанный продукт** сбрасывается в ёмкости или в атмосферу через коллектор.

---

### Паромы

**Маршрутный транспорт.**

- Соединяет фермерские поселения и города
- Малые паровые двигатели или МАГ
- Движение по системе подвесных тросов
- Максимальная скорость: 40 км/ч

## 📡 Технологии

### СОЛ — Система определения личности

Комплекс городских систем для идентификации граждан. Технология гравитационного сканирования.

### ГРАДАР

Комплекс определения координат физических тел в пространстве посредством гравитационного сканирования.

### ЛРО — Лётный радиолокационный ответ

Обмен техническими характеристиками и положением подвесных конструкций летательного аппарата.

### Алиенум

Определение для людей, потерявших память и не идентифицирующихся СОЛ.

### Фрейхейт (Фрей)

Совокупный образ неизвестного места, к которому стремятся герои.

##  Механики для геймплея

### Высота и облака

- **Облачный слой** — содержит пары мезия (яд)
- **Завеса** — искусственный дым сверху облаков
- **Горные пики** — выступают над облаками, на них поселения
- **Фермерские угодья** — опорные платформы с антигравием

### Топливо (мезий)

- Добывается из нижних слоёв (под облаками)
- Нужно для генераторов МАГ
- Ограниченный ресурс — планирование маршрутов
- **Риск:** спуск к мезию = опасность

### Энергия (электричество)

- Генераторы → рамка-контур → зависание
- Без энергии = падение
- Дрейф (без движения) потребляет минимум

### Движение

- Ветровые лопасти = горизонтальное движение
- Электродвигатели = управление лопастями
- Плавность — ключевая характеристика
