## 🎯 Архитектура (из ADR-0002)

### Компоненты Фазы 1:

```
WorldStreamingManager (Singleton)
├── WorldChunkManager → GetChunkAtPosition(), GetChunksInRadius()
├── ProceduralChunkGenerator → GenerateChunkAsync()
├── ChunkLoader → LoadChunk/UnloadChunk
└── FloatingOriginMP → ResetOrigin()
```

### Ключевые решения:
- Chunk Size: **2000x2000 units** (grid-based)
- Floating Origin threshold: **100,000 units**
- Генерация: **детерминированная из Seed** (server = client)
- Контент: горы/облака — НЕ NetworkObjects, фермы — NetworkObjects
