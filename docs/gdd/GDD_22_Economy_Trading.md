## 1. Overview

Экономическая система Project C: The Clouds включает **валюту (кредиты)**, **ресурсы** (мезий, антигравий, МНП, латекс), **торговлю** между игроками и NPC, **динамические цены** на основе спроса и предложения.

## 4. Pricing Model

### Формула цены

```
price = base_price × (1 + demand_factor - supply_factor) × reputation_mod × event_mod
```

| Множитель | Описание | Диапазон |
|-----------|----------|----------|
| `base_price` | Базовая цена ресурса | Фиксированная |
| `demand_factor` | Спрос в данной локации | -0.5 … +1.0 |
| `supply_factor` | Предложение в данной локации | -0.5 … +1.0 |
| `reputation_mod` | Модификатор репутации | 0.8 … 1.2 |
| `event_mod` | Модификатор событий | 0.5 … 3.0 |

### Пример расчёта

| Параметр | Значение |
|----------|----------|
| Ресурс | Мезий |
| Базовая цена | 10 CR |
| Спрос в Тертиусе | +0.3 (торговый hub, высокий спрос) |
| Предложение | -0.2 (много поставок) |
| Репутация игрока | +0.1 (дружелюбный) |
| Событие | 1.0 (нет событий) |
| **Итого** | 10 × (1 + 0.3 - 0.2) × 0.9 × 1.0 = **9.9 CR** |

## 6. Supply & Demand

### Динамика рынка

| Фактор | Влияние | Пример |

| Контрабанда мезия | ↓ Цена в Тертиусе | Много нелегального мезия |

## 10. Formulas

| Формула | Описание |
|---------|----------|
| `price = base × (1 + demand - supply) × rep_mod × event_mod` | Цена товара |
| `profit = sell_price - buy_price - tax` | Прибыль от торговли |
| `tax = trade_value × 0.05` | Налог на сделку (5%) |
| `cargo_penalty = cargo_weight / max_capacity` | Штраф скорости от груза |
| `contract_reward = base × distance × difficulty` | Награда за контракт |
