## 1. Overview

Нарративная система Project C: The Clouds основана на книге **«Интеграл Пьявица»** Бруно Арендта. Документ описывает историю мира, технологии, фракции, ключевых персонажей и способы интеграции лора в геймплей.

### Хронология

| Год | Событие | Описание |
|-----|---------|----------|
| **1930** | Мировое Падение | Метеориты несут антигравий и мезий на Землю |

### Мировое Падение (1930)

| Параметр | Описание |
|----------|----------|
| Причина | Падение метеоритов на Землю |
| Вещества | **Антигравий** (металл, нарушающий гравитацию) + **Мезий** (яд, t° кипения ≈ 17°C) |
| Последствия | Поверхность отравлена парами мезия |
| Решение | Искусственная Завеса — дым, сдерживающий пары |
| Эвакуация | ~2 млн человек на горные вершины |

## 4. Technology

### Ключевые технологии

| Технология | Описание | Применение |
|------------|----------|-----------|
| **МАГ/ГМАГ** | Мезий-антигравиевые генераторы | Электричество, антигравитация |
| **Антигравий** | Металл, нарушающий гравитацию при токе | Двигатели кораблей, опоры платформ |
| **Мезий** | Яд, t° кипения ≈ 17°C | Топливо, оружие, Завеса |
| **Ветровые лопасти** | Электродвигатели горизонтального движения | Корабли, паромы |
| **ГРАДАР** | Гравитационный радар навигации | Навигация кораблей |
| **СОЛ** | Система идентификации (гравитационное сканирование) | Контроль граждан |
| **Паромы** | Маршрутный транспорт по подвесным тросам | До 40 км/ч между пиками |
| **Рамка-контур** | Контур корабля → антигравитационное поле | Стабилизация кораблей |

### Физика технологий

| Принцип | Описание |
|---------|----------|
| Антигравитация | Ток через антигравий → нарушение гравитации |
| МАГ | Жидкий мезий + антигравиевый вал → электричество |
| Рамка-контур | Электрический контур → антигравитационное поле |
| Ветровые лопасти | Электричество → вращение → тяга |
