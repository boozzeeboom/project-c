## ⚠️ КРИТИЧНО: НЕ ТРОГАТЬ `.meta` ФАЙЛЫ

> **Никогда не создавать и не редактировать `.meta` файлы вручную!**

Unity автоматически создаёт `.meta` файлы для каждого ассета. Ручное создание/редактирование ломает ссылки, вызывает `TypeLoadException` и ошибки компиляции.

**Правило:**
- ✅ Unity создаёт `.meta` автоматически
- ✅ Не создавать, не удалять, не редактировать `.meta`
- ✅ Если случайно создал — удалить и дать Unity пересоздать

## 🎒 Архитектура инвентаря (решения сессии 4 апреля 2026)

### Круговой инвентарь (GTA-стиль)

**Принцип работы:**
- **Круговое колесо** — 8 секторов (ячеек), как выбор оружия в GTA 5
- **Ячейки закреплены за типами предметов** — Тип 1 → ячейка 1, Тип 2 → ячейка 2, и т.д.
- **Безлимитный** — нет ограничения на количество предметов
- **Автоматическая раскладка** — подобранный предмет сам падает в свою ячейку по типу
- **Группировка** — если подобрано несколько предметов одного типа, при наведении на ячейку раскрывается список всех предметов этого типа
- **Активация** — наведение курсора на сектор + выбор из списка (если несколько)

### Управление
| Клавиша | Действие | Режим |
|---------|----------|-------|
| **E** | Подобрать предмет (ближайший в радиусе) | Пеший режим |
| **Tab** | Открыть/закрыть круговой инвентарь | Пеший режим |
| **Q/E** | Лифт вверх/вниз | Режим корабля (не меняется) |

### Пример работы
```
Подобрал "Тип 1" → автоматически в ячейку 1
Подобрал ещё "Тип 1" → в ту же ячейку 1 (группировка)
Подобрал "Тип 3" → в ячейку 3
Открыл инвентарь (Tab) → навёл на ячейку 1 → раскрылся список: [Тип 1 x2]
```

### Визуальная схема (8 секторов)
```
        [0] Тип 1
    [7]         [1] Тип 2
  [6]    ⭕      [2] Тип 3
    [5]         [3] Тип 4
        [4] Тип 5
```
*На этапе тестирования: секторы 0-7 = Типы 1-8, пронумерованы*

### Технические компоненты
| Компонент | Назначение |
|-----------|------------|
| `ItemData.cs` | ScriptableObject — имя, тип, иконка, описание |
| `ItemType.cs` | Enum — типы предметов (1-8 для теста) |
| `PickupItem.cs` | Компонент подбираемого предмета (триггер) |
| `Inventory.cs` | Менеджер инвентаря (группировка по типам, список) |
| `InventoryUI.cs` | Круговое колесо (8 секторов, hover, подсписки) |
| `ItemPickupSystem.cs` | Связь: E → поиск ближайшего → подбор → инвентарь |

- Включен для: `*.png`, `*.jpg`, `*.fbx`, `*.wav`, `*.mp3`, `*.dll` и других больших файлов
- Перед коммитом больших файлов убедитесь, что они отслеживаются LFS: `git lfs track "*.extension"`

- Большие бинарные файлы без LFS
