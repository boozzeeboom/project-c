**Решение:** Добавлен `[ExecuteInEditMode]` в FarmPlatform, TerraceBuilder, FerryCableSystem.

### Проблема 3: Объекты слишком маленькие, «сбились в кучу»
**Решение:** Все размеры ×5 (см. таблицу ниже).

### Проблема 4: Ошибки CS0826 (implicitly-typed array)
**Решение:** `new[] { }` → `new string[0]`, `new[] { "a" }` → `new string[] { "a" }`.

**Решение:** Переименовано в `existingFarmsRoot`.

### Проблема 6: Баг collider троса (posA.x вместо posA.z)
**Решение:** Исправлен `Mathf.Abs(posB.z - posA.z)`.

## 🎯 КЛЮЧЕВЫЕ ТЕХНИЧЕСКИЕ РЕШЕНИЯ

### 1. Масштаб высот — В МЕТРАХ!
- **FarmData.worldPosition:** Y = 2500-4500 (метры), НЕ scaled units
- **Все 9 ферм** используют реальные метры
- **FarmPlacementEditor:** координаты зашиты в метрах

### 2. Модульная архитектура
```
FarmPlatform (root)
├── AntiGravPlatform — антигравийная плита (40x20x2)
├── Terraces (parent)
│   ├── Terrace_1 — 40x20
│   ├── Terrace_2 — 30x15
│   └── Terrace_3 — 20x10
├── Buildings (parent)
│   ├── Building_1, Building_2, Building_3
│   └── Greenhouse (если hasGreenhouse)
├── DockingBay — посадочная площадка 15x15
└── FerryCables (в корне Farms)
    ├── Cable_everest_010_to_everest_020
    └── ...
```

### 3. Runtime-генерация мешей
- **Mesh.CreateBox()** — Unity 6 API для создания бокс-мешей
- Все меши создаются программно при Start()
- НЕ требуются внешние префабы

### 4. Материалы — Editor-скрипт
- Materials создаются через `Tools → Project C → Farms → Create Farm Materials`
- 4 материала: Terrace, AntiGravPlatform, Greenhouse, FarmBuilding
- URP Lit/Unlit shaders

## 🚨 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Проблема 1: Mesh.CreateBox не найден

**Ошибка:** `CS0117: 'Mesh' does not contain a definition for 'CreateBox'`

**Решение:**
- Требуется Unity 6 (URP 14+)
- Если версия старая — нужно создать extension метод
- **НЕ КОММИТИТЬ** до решения проблемы

---

### Проблема 2: Материалы розовые

**Причина:** Shader не найден или материал не создан

**Решение:**
1. Запустить `Tools → Project C → Farms → Create Farm Materials`
2. Проверить что URP Lit/Unlit shaders доступны
3. В Project Settings → Graphics → Shader Preload

---

### Проблема 3: Фермы не видны в Scene View

**Причина:** Позиции в метрах (Y=2500-4500), камера далеко

**Решение:**
1. Выделите ферму в Hierarchy → нажмите F (focus)
2. Или используйте WorldCamera для полёта
3. Или уменьшите Y в FarmData для тестирования

---

### Проблема 4: Тросы не соединяют фермы

**Причина:** connectedFarmIds не назначены или ферма не найдена

**Решение:**
- Проверить FarmData каждой фермы
- Console: `[FerryCableSystem] stationA или stationB не назначены!`
