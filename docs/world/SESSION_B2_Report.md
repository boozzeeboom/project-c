## ⚠️ Известные проблемы

### 1. Визуальное несоответствие
**Проблема:** Столбы НЕ похожи на кумуло-дождевые облака из книги.

**Текущий вид:**
- Полупрозрачный фиолетовый цилиндр
- Нет формы "наковальни" (расширение к вершине)
- Нет плотности/объёмности облака
- Молнии — простые частицы, не фиолетовые вспышки

**Ожидание (из книги):**
- Массивные вертикальные столбы тёмных туч
- Расширение к вершине (наковальня)
- Фиолетовые молнии бьют внутри столба
- Нижняя часть столба фиолетовая от завесы

**Статус:** ⏳ Отложено — текущая реализация может быть основой для дальнейшего улучшения

### 2. Технические проблемы (РЕШЕНЫ)

#### BUG-001: Краш Unity — Resource ID out of range
**Симптомы:**
- Ошибка: `Resource ID out of range in GetResource: 1068745 (max is 1048575)`
- Unity крашился при Play

**Корневая причина:**
1. Процедурный меш с огромными размерами (vertexCount > 1000)
2. Particle System с масштабом 1280x7800x1280
3. Unity не справляется с Resource ID для огромных мешей

**Решение:**
1. ✅ Заменён процедурный меш на стандартный Unity Cylinder (GameObject.CreatePrimitive)
2. ✅ Уменьшены сегменты меша (24→12, 16→8)
3. ✅ Particle System масштаб уменьшен до 100x100x100
4. ✅ Уменьшено количество частиц (20→5)
5. ✅ Увеличен интервал молний (15-45s → 30-90s)
6. ✅ Упрощён TriggerLightning (1 частица вместо 2-5)

**Результат:** ✅ Unity НЕ крашится, стабильная работа
