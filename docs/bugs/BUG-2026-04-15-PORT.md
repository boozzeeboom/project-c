# BUG: Port 7777 Already in Use

## Summary
При попытке запустить Host возникает ошибка `Failed to bind UDP socket because the address is already in use` на порту 7777.

## Severity
**P3** — Workaround exists

## Steps to Reproduce
1. Запустить Unity Editor
2. Нажать "Start Host" в NetworkUI
3. Ошибка: "Failed to bind UDP socket because the address is already in use"

## Root Cause
Порт 7777 занят предыдущим процессом Unity, который не освободил сокет (TIME_WAIT).

## Solutions

### Solution 1: Wait (Recommended)
Подождать 10-30 секунд пока операционная система освободит порт.

### Solution 2: Change Port
1. Edit → Project Settings → Unity Transport
2. Изменить Default IP Address порт: `7777` → `7778`
3. Также изменить клиентский адрес подключения

### Solution 3: Kill Process (Windows)
```cmd
netstat -ano | findstr :7777
taskkill /PID <process_id> /F
```

## Files Affected
None — это системная проблема, не связана с кодом.

## Prevention
- Использовать разные порты для разных тестовых сессий
- Добавить автоинкремент порта в NetworkUI
