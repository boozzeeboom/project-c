# 🎮 Управление в Project C

## 🌍 Навигация по миру (камера)

### Горячие клавиши:

| Клавиша | Действие |
|---------|----------|
| **W / S** | Движение вперёд/назад |
| **A / D** | Стрейф влево/вправо |
| **Мышь** | Вращение камеры |
| **Left Shift** | Ускорение (до 500 м/с) |
| **F** | Переключить режим полёта |
| **N / PageUp** | Телепорт к следующему пику ⬆️ |
| **B / PageDown** | Телепорт к предыдущему пику ⬇️ |
| **R** | Телепорт к случайному пику 🎲 |
| **H** | Возврат на высоту (2000м) ⬆️ |
| **Колёсико мыши** | Набор высоты/снижение |

---

## 🏔️ Стартовая позиция

При запуске сцены:
- Камера появляется на высоте **5000м**
- Автоматически телепортируется к **первому пику**
- Вы видите горную вершину перед собой

---

## 📍 Горные пики (точки интереса)

Параметры пиков:
- **Количество:** 15
- **Высота:** 1000-8000м над облаками
- **Радиус основания:** 200-800м
- **Распределение:** по спирали (равномерно по миру)

---

## ☁️ Облачный слой

- **Высота:** 200м от "дна" мира
- **Толщина:** 100м
- **Покрытие:** 70% мира
- Игра происходит **над облаками**

---

## 🎯 Советы по навигации

1. **Быстрый осмотр мира:**
   - Нажмите **R** для телепорта к случайному пику
   - Используйте **N/B** для последовательного осмотра

2. **Детальный осмотр пика:**
   - Отключите режим полёта (**F**)
   - Приземлитесь на вершину
   - Включите режим полёта снова для отлёта

3. **Потерялись?**
   - Нажмите **H** для возврата на высоту
   - Оттуда видно все пики

4. **Скорость:**
   - Обычная: 100 м/с
   - Ускорение (Shift): 500 м/с

---

## 🔧 Настройки в Unity Inspector

Для **WorldCamera**:
- **Start Height:** начальная высота (по умолчанию 5000м)
- **Fly To First Peak On Start:** авто-телепорт к первому пику
- **Fly Speed:** обычная скорость (100 м/с)
- **Max Speed:** скорость при ускорении (500 м/с)
- **Mouse Sensitivity:** чувствительность мыши

Для **WorldGenerator**:
- **World Radius:** радиус мира (10км)
- **Peak Count:** количество пиков (15)
- **Min/Max Peak Height:** диапазон высот пиков
- **Cloud Layer Height:** высота облаков (200м)
