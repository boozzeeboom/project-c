## 3. Large Scale MMO — Документация

### Расположение
```
docs/world/LargeScaleMMO/
└── SUMMARY_2026-04-16.md  ← Основной документ
```

### Суть системы
Мир более 100 000 единиц требует Floating Origin — игрок всегда near (0,0,0), мир смещается вокруг него.

### Что сделано (Phase 1 ✅)
- `FloatingOriginMP.cs` — основная система смещения
- `WorldStreamingManager.cs` — управление загрузкой чанков
- `WorldChunkManager.cs` — менеджер чанков
- `ChunkLoader.cs` — загрузчик чанков
- `ProceduralChunkGenerator.cs` — процедурная генерация

### Что осталось (Phase 2 🔄)
- Runtime streaming — загрузка/выгрузка чанков в реальном времени
- Persistence — сохранение состояния объектов
- NPC Spawning — спавн NPC по чанкам
- Async Loading — асинхронная загрузка

### Roadmap
```
Phase 1 ✅ (v1.0.0)     — Foundation: FloatingOriginMP, basic chunking
Phase 2 🔄 (v1.1.0)     — Runtime streaming, persistence
Phase 3 📋 (v1.2.0)     — Optimization: LOD, async loading
Phase 4 📋 (v2.0.0)     — Full MMO: 1000+ players per shard
```
