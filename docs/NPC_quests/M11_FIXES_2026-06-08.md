# M11 — Исправления Dialog Flow (2026-06-08)

## Проблема 1: Видимость "Ну что, нашёл Кристалл?" без квеста
**Причина:** `QuestStateEquals` возвращал `true` когда квеста нет в логе.
**Фикс:** `return false` когда quest не найден в `GetPlayerQuests()`.
**Результат:** Edge невидим (hideIfUnavailable + !available = фильтрация).

## Проблема 2: Нужен ручной Accept в P → КВЕСТЫ
**Причина:** `OfferQuest` (type=10) ставит квест в Discovered.
**Решение:** Добавлен новый action `AcceptQuest = 14`.
- `QuestServer.FireDialogAction.AcceptQuest`: вызывает `TryOffer` (idempotent) + `TryAccept` → квест сразу в Active.
- Dialog tree: edge "Хорошо" → `type: 14` вместо `type: 10`.

**Результат:** "Я помогу" → TakeItem(A) → "Хорошо" → AcceptQuest → Active. Без P menu.

## Проблема 3: Награды не работали
**Причина:** Quest был в Discovered → `CompleteObjective` → `TryTurnIn` → fail (нужен Active/Completed).
**Фикс:** после AcceptQuest quest теперь Active → `TryTurnIn` работает (Active→auto Complete→TurnedIn).
**Результат:** `complete_thanks` onEnterActions (AddRep 25, AddAtt 10, CompleteObjective) + edge (GiveCredits 1000) — всё срабатывает.
