## 🔴 Обнаруженные баги

### БАГ #1 (КРИТИЧЕСКИЙ) — FloatingOriginMP разрушал иерархию сцены

**Файл:** `ThirdPersonCamera.cs`

**Проблема:** `Awake()` безусловно добавлял `FloatingOriginMP` на камеру:
```csharp
private void Awake()
{
    gameObject.AddComponent<FloatingOriginMP>(); // ← УДАЛЕНО
    // ...
}
```

**Цепочка破坏и:** 
1. FloatingOriginMP.Awake() → FindOrCreateWorldRoots()
2. FindOrCreateWorldRoots() не находил "Mountains"/"Clouds"/"WorldRoot"
3. CollectWorldObjects() → scene.GetRootGameObjects()
4. ВСЕ root-объекты сцены репарентились под "WorldRoot"
5. NetworkPlayer, NetworkManager — всё сломалось

**Исправление:**
- Убран вызов `AddComponent<FloatingOriginMP>()` из ThirdPersonCamera
- FloatingOriginMP больше не создаётся автоматически

---

### БАГ #2 — NetworkPlayer спавнил камеру как child

**Файл:** `NetworkPlayer.cs`

**Проблема:**
```csharp
var camObj = Instantiate(cameraPrefab.gameObject, transform); // child!
```

**Исправление:**
```csharp
var camObj = Instantiate(cameraPrefab.gameObject); // standalone
camObj.name = $"ThirdPersonCamera_{OwnerClientId}";
```

---

### БАГ #3 — ThirdPersonCamera не лочила курсор

**Файл:** `ThirdPersonCamera.cs`

**Проблема:** После InstantiateCamera() не было `Cursor.lockState = CursorLockMode.Locked`

**Исправление:** Добавлен `InitializeCamera()` с проверкой `NetworkManager.Singleton.IsListening`:
```csharp
public void InitializeCamera()
{
    if (_cameraInitialized) return;
    if (target == null) { Debug.LogWarning(...); return; }
    _yaw = 0f; _pitch = 15f;
    _currentDistance = distance; _currentHeight = height;
    bool inActiveGame = NetworkManager.Singleton != null && NetworkManager.Singleton.IsListening;
    if (inActiveGame) { Cursor.lockState = CursorLockMode.Locked; Cursor.visible = false; }
    else { Cursor.lockState = CursorLockMode.None; Cursor.visible = true; }
    UpdateCameraPosition();
    CreateControlHintsUI();
    _cameraInitialized = true;
}
```

Добавлен `OnDestroy()` для разблокировки курсора:
```csharp
private void OnDestroy()
{
    Cursor.lockState = CursorLockMode.None;
    Cursor.visible = true;
}
```

### БАГ #4 — WorldCamera безусловно лочила курсор

**Файл:** `WorldCamera.cs`

**Проблема:** Безусловный `Cursor.lockState = CursorLockMode.Locked` в Start() блокировал меню

**Исправление:**
```csharp
private void Start()
{
    // Больше не лочим курсор — ThirdPersonCamera управляет этим
    Cursor.lockState = CursorLockMode.None;
    Cursor.visible = true;
}
```

**Проблема:** При отсутствии world roots создавал новый WorldRoot и репарентил ВСЁ

if (worldRoots.Length == 0)
