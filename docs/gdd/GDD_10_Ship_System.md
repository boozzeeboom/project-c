## 1. Overview

Система кораблей — **ключевая механика** Project C: The Clouds. Корабли на антигравии — это не истребители, а медленные баржи с плавной физикой, инерцией и стабилизацией. Несколько игроков могут пилотировать один корабль совместно.

### Ключевые особенности
- **Антигравитация** — полная компенсация гравитации, зависание
- **Ветровые лопасти** — горизонтальная тяга
- **Нет крена** — рамка-контур стабилизирует
- **Кооп-пилотирование** — усреднение ввода через ServerRpc

## 3. Flight Physics

### Физика полёта

| Сила | Формула | Описание |
|------|---------|----------|
| **Антигравитация** | `F_grav = -g * mass` | Полная компенсация гравитации |
| **Тяга** | `F_thrust = forward * inputZ * baseThrust * boost` | Ветровые лопасти |
| **Сопротивление** | `F_drag = -velocity * drag` | Замедление без тяги |
| **Стабилизация** | `rot = Slerp(current, horizontal, stabSpeed * dt)` | Возврат к горизонту |
| **Рыскание** | `torque_y = inputX * yawSpeed` | Поворот корпуса |
| **Тангаж** | `torque_x = mouseY * pitchSpeed` | Наклон носа |
| **Лифт** | `F_lift = up * (inputQ - inputE) * liftSpeed` | Вертикальное движение |

### Принципы физики

1. **Медленный разгон** — масса + инерция
2. **Плавное зависание** — антигравитация компенсирует вес
3. **Нет крена** — рамка-контур стабилизирует
4. **Автоматическая стабилизация** — возврат к горизонту
5. **Инерция массы** — тяжёлые корабли медленнее разгоняются и тормозят

## 4. Controls

### Управление кораблём

| Клавиша | Ось | Действие | Параметр |
|---------|-----|----------|----------|
| W | +Z | Тяга вперёд | `baseThrust` |
| S | -Z | Тяга назад | `baseThrust * 0.5` |
| A | -Y | Рыскание влево | `yawSpeed = 60°/s` |
| D | +Y | Рыскание вправо | `yawSpeed = 60°/s` |
| Q | -Y (локально) | Лифт вниз | `liftSpeed = 10 м/с` |
| E | +Y (локально) | Лифт вверх | `liftSpeed = 10 м/с` |
| Мышь Y | X (локально) | Тангаж | `pitchSpeed = 45°/s` |
| Left Shift | Boost | Ускорение x2 | `boostMultiplier = 2.0` |
| F | Action | Выйти из корабля | height < 3м, speed < 1м/с |

## 5. Co-op Piloting

### Кооперативное пилотирование

| Аспект | Описание |
|--------|----------|
| **Механика** | Несколько игроков (2+) в одном корабле |
| **Усреднение** | `finalInput = Sum(pilotInput[i]) / pilotCount` |
| **Серверная авторитетность** | SubmitShipInputRpc → сервер усредняет → репликация |
| **Роли** | [🔴 Запланировано] Капитан, штурман, инженер |
| **Приоритет** | [🔴 Запланировано] Капитан имеет больший вес (1.5x) |

### Сетевая синхронизация

| Метод | Описание |
|-------|----------|
| `SubmitShipInputRpc()` | Клиент шлёт ввод на сервер |
| `AddPilotRpc()` | Добавление пилота, синхронизация всем |
| `RemovePilotRpc()` | Удаление пилота, синхронизация всем |
| ServerAuthority | NetworkTransform — сервер единственный источник истины |

## 7. Boarding/Disembarking

### Посадка в корабль

| Условие | Значение |
|---------|----------|
| Расстояние | < 5м до ближайшего корабля |
| Скорость корабля | < 1 м/с |
| Режим игрока | Пеший (Walking) |
| Действие | Нажать F |

### Выход из корабля

| Условие | Значение |
|---------|----------|
| Высота корабля | < 3м над поверхностью |
| Скорость корабля | < 1 м/с |
| Режим игрока | Полёт (Flying) |
| Действие | Нажать F |

### Переключение камеры

| Событие | Изменение |
|---------|-----------|
| Посадка | Camera distance: 5м → 18м, target → корабль |
| Выход | Camera distance: 18м → 5м, target → игрок |

## 9. Edge Cases

| Ситуация | Поведение | Реализация |
|----------|-----------|-----------|
| **Все пилоты вышли в воздухе** | Корабль медленно снижает скорость, затем зависает | ✅ ShipController |
| **Посадка в движущийся корабль** | Отказ, проверка speed < 1 м/с | ✅ PlayerStateMachine |
| **Выход на большой высоте** | Отказ, проверка height < 3м | ✅ PlayerStateMachine |
| **Столкновение с пиком** | Отскок по физике Rigidbody | ✅ Физика Unity |
| **Дисконнект пилота** | Пилот удалён, корабль продолжает | ✅ NetworkManagerController |
| **Корабль перевернулся** | [🔴 Запланировано] Авто-выравнивание | — |
| **Boost не передаётся в RPC** | [⚠️ Известная проблема] Добавить параметр в SubmitShipInputRpc | 🔴 |
