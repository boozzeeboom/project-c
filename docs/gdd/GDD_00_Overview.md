## 1. Overview

### Game Concept

**Project C: The Clouds** — приключенческая MMO-игра в открытом мире над облаками. 2090 год, человечество ютится на горных вершинах, окутанных Завесой — ядовитым слоем, скрывающим отравленную поверхность. Игроки управляют пилотами антигравийных кораблей-барж, курсирующих между городами и фермами, выполняя контракты, исследуя заброшенные платформы и взаимодействуя с фракциями.

Игра основана на книге **«Интеграл Пьявица»** Бруно Арендта. Мир тоталитарного контроля, пяти гильдий, подпольных организаций и технологий на основе антигравия и мезия.

### Design Pillars

| # | Пиллар | Описание | Пример реализации |
|---|--------|----------|------------------|
| 1 | **Свобода полёта** | Корабли — не истребители, а баржи. Плавный, медитативный полёт с инерцией и стабилизацией | ShipController: антигравитация, медленный разгон, нет крена |
| 2 | **Исследование над боем** | Основной геймплей — исследование мира, а не сражения. Бой — через стелс и избегание | Завеса как опасность, СОЛ-система обхода |
| 3 | **Кооперативное взаимодействие** | Несколько игроков в одном корабле, разделение ролей, общая цель | Кооп-пилотирование: усреднение ввода через ServerRpc |
| 4 | **Sci-Fi + Ghibli эстетика** | Промышленный дизайн + мягкие облака, градиенты, объёмный свет | CloudGhibli.shader, URP, цветовая палитра |
| 5 | **Всё на технологии, ничего магического** | Антигравий, мезий, МАГ — научно объяснимые технологии | WORLD_LORE_BOOK.md: физика антигравия |

### Target Audience

| Сегмент | Описание | Мотивация |
|---------|----------|-----------|
| **Исследователи** (основная) | Любят открывать новые места, лор, секреты | Процедурный мир, заброшенные платформы, лор-объекты |
| **Кооп-игроки** | Играют с друзьями, ценят совместный геймплей | Кооп-корабли, общие миссии, разделение ролей |
| **Фанаты лора** | Читали книгу, хотят погрузиться в мир | Фракции, квесты, диалоги, environmental storytelling |
| **Казуальные** | Расслабленный геймплей, медитативный полёт | Плавная физика, красивый визуал, нет давления |

### Unique Selling Points (USP)

1. **Мир над облаками** — уникальная вертикальная навигация, 3D-пространство без «земли»
2. **Антигравийные баржи** — не истребители, а медленные, инерционные корабли с физикой
3. **Ghibli-эстетика в Sci-Fi** — мягкие облака + промышленный дизайн, редкое сочетание
4. **Кооп-пилотирование** — несколько игроков управляют одним кораблём
5. **Литературная основа** — глубокий лор из книги «Интеграл Пьявица»

### Core Fantasy

> *«Ты — свободный пилот над облаками. Мир огромен, города парят на вершинах, а Завеса скрывает тайны поверхности. Твой корабль — твой дом, твой инструмент для исследования и выживания в мире тоталитарного контроля.»*

## 2. Player Fantasy

### Что чувствует игрок

| Ситуация | Ощущение | Как достигается |
|----------|----------|----------------|
| Первый полёт | **Свобода и простор** | Облака под ногами, бескрайнее небо, плавное ускорение |
| Посадка на заброшенной платформе | **Одиночество и тайна** | Тихий ветер, ржавые конструкции, лут в сундуках |
| Полёт через Завесу | **Опасность и напряжение** | Фиолетовые молнии, низкая видимость, риск |
| Кооп-пилотирование с другом | **Командная работа** | Один рулит, второй сканирует, общие решения |
| Обход СОЛ-патруля | **Стелс и хитрость** | Выключение транспондера, обходной маршрут |

### Эмоциональная кривая (сессия)

```
Исследование (спокойствие) → Контракт (фокус) → Находка (радость) → 
Опасность (напряжение) → Возвращение (расслабление) → Прогресс (удовлетворение)
```

## 3. Detailed Rules

### Основные правила геймплея

1. **Два режима существования:**
   - Пеший режим (вид от третьего лица): города, платформы, NPC, сундуки
   - Режим корабля (полёт): перемещение между локациями, контракты, исследование

2. **Переключение режимов:**
   - Клавиша **F**: сесть в ближайший корабль (< 5м) / выйти из корабля
   - Камера автоматически адаптируется

3. **Ресурсы:**
   - **Кредиты** — основная валюта (контракты, торговля)
   - **Мезий** — топливо кораблей (расходуется, нужна заправка)
   - **Антигравий** — компонент ремонта/улучшений
   - **МНП** — мезий-антигравиевый препарат (future)
   - **Латекс** — технический ресурс (future)

4. **Фракции:**
   - 5 Гильдий: репутация, ранги, квесты, награды
   - Подполье: скрытые квесты, контрабанда, сопротивление
   - Правительство: СОЛ, патрули, преследование

5. **Смерть/провал:**
   - Падение в Завесу → телепорт на ближайший пик + штраф
   - Столкновение → потеря груза + ремонт
   - [🔴 Запланировано] Система здоровья

## 4. Formulas

### Базовые формулы (текущие)

| Формула | Описание |
|---------|----------|
| `thrust = baseThrust * (boost ? 2.0f : 1.0f)` | Тяга корабля с бустом |
| `gravity_compensation = -Physics.gravity * rigidbody.mass` | Антигравитация |
| `stabilization = Quaternion.Slerp(current, target, speed * dt)` | Стабилизация корабля |
| `cooperative_input = Sum(pilot_input[i]) / pilot_count` | Усреднение ввода |
| `movement_speed = baseSpeed * (isRunning ? 2.0f : 1.0f)` | Скорость пешком |

### Базовые параметры (настраиваемые)

| Параметр | Значение | Описание |
|----------|----------|----------|
| `baseThrust` | 50.0 | Базовая тяга корабля |
| `drag` | 0.5 | Сопротивление воздуха |
| `stabilizationSpeed` | 2.0 | Скорость стабилизации |
| `boardingRadius` | 5.0 | Радиус посадки в корабль |
| `pickupRadius` | 3.0 | Радиус подбора предметов |
| `maxShipSpeed` | [зависит от класса] | Максимальная скорость |

## 5. Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| **Игрок упал в Завесу** | Телепорт на ближайший пик + штраф (потеря части груза) |
| **Дисконнект в полёте** | Корабль зависает, инвентарь сохранён, авто-реконнект 5 попыток |
| **Все пилоты вышли из корабля в воздухе** | Корабль медленно снижается (автопилот), затем зависает |
| **Игрок застрял на пике** | Teleport fallback через WorldCamera (V+R) |
| **Корабль столкнулся с пиком** | Отскок + минимальный урон (пока нет системы здоровья) |
| **Два игрока подбирают предмет одновременно** | Сервер определяет порядок, первый получает предмет |

## 8. Acceptance Criteria

| # | Критерий | Как проверить |
|---|----------|--------------|
| 1 | Игрок может ходить/бегать/прыгать | WASD + Shift + Space в пешем режиме |
| 2 | Игрок может сесть/выйти из корабля | F, радиус < 5м, проверка скорости |
| 3 | Корабль летает плавно, с инерцией | WASD + мышь, отсутствие резких рывков |
| 4 | Кооп-пилотирование работает | 2+ игрока, ввод усредняется |
| 5 | Камера адаптируется при смене режима | Плавный переход distance 5м → 18м |
| 6 | Инвентарь работает (подбор, сундуки) | E подбирает, Tab открывает колесо |
| 7 | Сетевой мультиплеер работает | Host + Client, синхронизация движения |
| 8 | Reconneкт работает | Обрыв → авто-подключение, инвентарь сохранён |
| 9 | Визуал соответствует Ghibli-стилю | Облака с rim glow, мягкие цвета |
| 10 | Core Loop работает | Контракт → полёт → выполнение → награда |
