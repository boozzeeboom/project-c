### Итерация 2-Б: Tails of the Fox

**Фокус:** 
Добавление персонажа, предположительно лиса. Небольшая игра “исследование и коллекционирование”. Итерация предназначена как переходная между 1 и 2-А для отлаживания, изучения и отработки будущих механик и анимации. Планируется подробное изучение вариаций процедурной генерации в world partition, locomotion, UI дизайне, RPG систем прокачки и взаимодействия с миром, строительства и много другого.

**Ключ: Изучить-> отработать->отсеять лишнее->оставить для Итерации 2-А**

**Основные цели:**
 - [ ] Отработать систему locomotion для персонажей бег-полет.
 - [ ]  отработать способы наполнения мира
 - [ ] система персонаж-тс-инвентарь для отображения всех предметов.
 - [ ] строительство\размещение\передвижение объектов.
 - [x] мультиплеерная составляющая для 2х игроков.
 - [ ] подобие RPG системы инвентаря-предметов-навыков.
 - [ ] генерация объектов коллекционирования.

**Геймплей:**

- [ ] исследование мира с целью поиска предметов и наполнение ими персонажа и “точки дома”.

- [ ] взаимодействие в мультиплеерном режиме.

- [ ] примитивные головоломки.
