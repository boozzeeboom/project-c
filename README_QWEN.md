## ⚙️ Настройки Git

### Git LFS (Large File Storage)

Включен для автоматической загрузки больших файлов:
- Изображения: `*.png`, `*.jpg`, `*.tga`, `*.psd`
- 3D модели: `*.fbx`, `*.obj`, `*.blend`
- Аудио: `*.wav`, `*.mp3`, `*.ogg`
- DLL: `*.dll`

**Проверка статуса LFS:**
```bash
git lfs status
git lfs ls-files
```

### Unity Git Integration

У вас **включена Git-синхронизация в Unity Editor**. Это означает:

✅ **Можно:**
- Делать коммиты прямо из Unity
- Видеть изменения файлов в реальном времени

⚠️ **Важно:**
- После коммитов из Unity делайте `git push` в терминале
- Большие файлы (PNG, текстуры) автоматически идут через LFS

## 📞 Если что-то пошло не так

### 1. Проверьте ветку Git:
```bash
git branch                      # Текущая ветка
git checkout qwen-code-f3db...  # Переключиться на рабочую ветку
```

### 2. Получите последние изменения:
```bash
git fetch upstream
git pull upstream qwen-code-f3dbb381-a66e-4c3f-8475-937fd11685bb
```

### 3. Отправьте изменения на GitHub:
```bash
git add .
git commit -m "Описание изменений"
git push upstream qwen-code-f3dbb381-a66e-4c3f-8475-937fd11685bb
```

### 4. Если Unity ругается на .meta файлы:
- Удалите проблемный `.meta` файл
- Unity автоматически пересоздаст его с правильным GUID
