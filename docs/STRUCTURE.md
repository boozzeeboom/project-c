## 📝 Краткое резюме проекта

**Project C: The Clouds** — MMO/Co-Op авиасимулятор по мотивам книги «Интеграл Пьявица» (Бруно Арендт).

**Мир:** 1930 — Мировое Падение → ~2090 — Новая Цивилизация (~2 млн чел.) на горных пиках над облаками. 4 города: Примум (Эверест), Секунд, Тертиус (Аконкагуа), Квартус (Мак-Кинли). Фермерские угодья по хребтам (ветки: ~15 основных, ~10 побочных).

**Технологии:** Антигравий + Мезий → МАГ/ГМАГ генераторы → рамка-контур → ветровые лопасти. Корабли — медленные баржи, без крена, плавная физика.

**Фракции:** 5 Гильдий (Мысли, Созидания, Силы, Тайн, Успеха) + подпольные организации + тоталитарное НП.

**Текущий статус (v0.0.5.1):** Процедурный мир (15 пиков + 890+ облаков), пешеходный контроллер, ShipController (Rigidbody), переключение F, ThirdPersonCamera, базовый Netcode.

**Стек:** Unity 6, URP, Input System, Netcode for GameObjects, .NET 8 сервер.
