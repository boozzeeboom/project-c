### Architecture Decisions Made
1. **WindManager as central singleton** — Single source of truth, all cloud systems subscribe via event
2. **ServerWeatherController as NetworkBehaviour** — Uses ClientRpc to broadcast, 0.5Hz interval
3. **Player-centered recycling** — Clouds recycle when >15km from player, regenerate at 8-12km
4. **World-space distant impostors** — Positions relative to world origin (0,0,0), not player-relative (fixes multiplayer desync)
5. **NearCloudRenderer per altitude layer** — Upper (6000-8000m, 80), Middle (3000-5000m, 120), Lower (1500-3000m, 80)
