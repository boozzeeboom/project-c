## 🔗 Архитектура

```
WorldStreamingManager (координатор)
    │
    ├── WorldChunkManager (реестр чанков)
    ├── ProceduralChunkGenerator (генерация контента)
    ├── ChunkLoader (загрузка/выгрузка)
    │       └── OnChunkLoaded/OnChunkUnloaded events
    └── PlayerChunkTracker (серверный трекинг)
            └── RPC: LoadChunkClientRpc / UnloadChunkClientRpc

FloatingOriginMP (дополнительно)
    └── Для краевых случаев (>1M units)
```

## ⚠️ Возможные проблемы

### 1. Компоненты не подключены в сцене
- FloatingOriginMP добавлен в сцену
- PlayerChunkTracker может быть не добавлен
- WorldStreamingManager может быть не настроен

### 2. FloatingOriginMP конфликтует с Chunk Streaming
- FloatingOriginMP сдвигает мир вокруг игрока
- Chunk Streaming загружает чанки вокруг игрока
- Обе системы могут конфликтовать

### 3. PlayerChunkTracker не инициализируется
- Требует NetworkManager
- Требует NetworkPlayer компонент
