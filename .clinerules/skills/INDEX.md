```

/code-review Assets/_Project/Scripts/Player
/sprint-plan new
/project-stage-detect
/tech-debt
```

## Добавление нового skill

1. Создать папку: `.clinerules/skills/[skill-name]/`
2. Добавить файл: `SKILL.md`
3. Добавить запись в этот INDEX.md
